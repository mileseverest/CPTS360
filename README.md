# CPTS360

Completely working EXT2 File system that runs with virtual disk "disk."

Includes commands: 

              -------  LEVEL 1 ------------ 
               mount_root;
               mkdir, rmdir, ls, cd, pwd;
               creat, link,  unlink, symlink
               stat,  chmod, touch;

              -------  LEVEl 2 -------------
               open,  close,  read,  write
               lseek  cat,    cp,    mv

              -------  LEVEl 3 ------------ 
               mount, umount
               File permission checking
              -----------------------------
